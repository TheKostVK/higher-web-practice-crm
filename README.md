## Запуск проекта

### Фронтенд

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу из вывода команды (обычно http://localhost:5173).

### Моковый бэкенд (json-server)

В отдельном терминале:

```bash
npm run server
```

Сервер поднимается на порту 3001 (или как указано в конфигурации). Базовый URL API использовать в настройках RTK Query (например, `http://localhost:3001`).

### Сборка

```bash
npm run build
```

### Линтинг

```bash
npm run lint
```
