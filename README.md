# YaPlex CRM

Учебная CRM-система для управления клиентами, сделками, задачами и отчётами. В проекте реализованы авторизация,
регистрация, dashboard со сводной статистикой, списки клиентов/сделок/задач с модальными формами и раздел отчётов с
фильтрами.

## Стек

- React
- TypeScript
- Redux Toolkit и RTK Query
- React Router
- React Hook Form и Zod
- Ant Design
- Vite
- Vitest и Testing Library

## Тестовые пользователи

```text
manager1@crm.ru / 123456
manager2@crm.ru / 123456
```

## Реализованные страницы

| Маршрут                 | Страница              | Основной сценарий                                                                                        |
| ----------------------- | --------------------- | -------------------------------------------------------------------------------------------------------- |
| `/`                     | Dashboard             | Просмотр сводной статистики, активных сделок, последних задач и топ-клиентов.                            |
| `/clients`              | Клиенты               | Просмотр, поиск, сортировка, создание, редактирование и удаление клиентов через модальные окна.          |
| `/deals`                | Сделки                | Просмотр, поиск, фильтрация, создание, редактирование, завершение и удаление сделок.                     |
| `/tasks`                | Задачи                | Просмотр, поиск, фильтрация, создание, редактирование, завершение и удаление задач.                      |
| `/reports`              | Отчёты                | Просмотр отчётов по продажам, этапам сделок, новым клиентам, активности клиентов и просроченным задачам. |
| `/profile`              | Профиль               | Редактирование данных пользователя, аватара и пароля.                                                    |
| `/auth`                 | Welcome               | Стартовая auth-страница.                                                                                 |
| `/auth/login`           | Вход                  | Авторизация тестового пользователя.                                                                      |
| `/auth/registration`    | Регистрация           | Создание нового пользователя с проверкой обязательных полей.                                             |
| `/auth/forgot-password` | Восстановление пароля | Проверка email-формы восстановления.                                                                     |
| `/auth/confirm-email`   | Подтверждение email   | Проверка ссылки подтверждения email.                                                                     |

## Запуск проекта

### Фронтенд

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу из вывода команды (обычно http://localhost:5173).

### Переменные окружения

Можно переопределить адрес mock API через `.env.local`:

```bash
VITE_API_BASE_URL=http://localhost:3001
```

### Моковый бэкенд (json-server)

В отдельном терминале:

```bash
npm run server
```

Сервер поднимается на порту 3001. Если переменная `VITE_API_BASE_URL` не задана, RTK Query использует
`http://localhost:3001`.

### Сборка

```bash
npm run build
```

### Тесты

```bash
npm test
```

### Линтинг

```bash
npm run lint
```
